# Stand/Sit Reminder

**Stand/Sit Reminder** is a lightweight Windows utility that helps you maintain a healthy routine by reminding you to **stand up** or **sit down** at regular intervals. The app runs quietly in the system tray and uses Windows toast notifications for alerts.

---

## Features

* **Customizable reminders:** Choose intervals of 15, 30, 60, or 120 minutes.
* **Toast notifications:**

  * **OK** – dismiss the reminder.
  * **Remind in 5 min** – snooze the alert.
* **Start with Windows:** Optional auto-start to always have reminders.
* **Tray menu:** Easily adjust interval settings, toggle auto-start, or exit the app.
* **Silent & lightweight:** Minimal system resources used.

---

## Installation

1. Download the **Stand/Sit Reminder ZIP file**.
2. Extract the contents to a folder of your choice.
3. Run the executable (`StandSitReminder.exe`). A tray icon will appear in your taskbar.
4. **Important:** Windows will ask if you want to allow the app to show urgent notifications. Click **Yes** to ensure reminders work even when **Do Not Disturb** is enabled.
5. The first reminder will appear immediately.

---

## Usage

* **Tray icon menu:**
  Right-click or left-click the tray icon to access:

  * **Interval selection** – set reminder frequency.
  * **Start with Windows** – enable or disable auto-start.
  * **Exit** – close the app.
* **Notifications:**

  * Click **OK** to dismiss.
  * Click **Remind in 5 min** to snooze the reminder.

---

## System Requirements

* Windows 10 or later.
* No installation required; runs as a standalone executable.
* Uses native Windows notifications.
