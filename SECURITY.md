# Security Policy

## 🔒 Supported Versions

Currently, only the **latest released version** of *Stand/Sit Reminder* is supported with security updates.

Older versions may not receive fixes.

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, **do not report it publicly**.

Instead, please report it responsibly:

📧 **Email:** [lexyfischer@web.de](mailto:lexyfischer@web.de)

Include the following details:

* Description of the vulnerability
* Steps to reproduce (if possible)
* Potential impact
* Screenshots or proof-of-concept (if applicable)

---

## 🤝 Responsible Disclosure

* Please allow reasonable time for the issue to be investigated and fixed before any public disclosure
* Do not share or publish the vulnerability details until a fix is released

---

## ⚠️ Scope

This policy applies to:

* The official **Stand/Sit Reminder executable**
* Notifications and system tray behavior
* Startup/auto-start functionality

Out of scope:

* Modified or unofficial versions of the software
* Third-party tools or environments

---

## 🛡️ What to Expect

After submitting a report:

* You will receive confirmation that your report was received
* The issue will be reviewed and validated
* A fix will be developed if necessary
* You may be contacted for additional details

---

## ❌ What Not to Do

* Do not attempt to exploit vulnerabilities beyond proof-of-concept
* Do not access or modify data that is not your own
* Do not perform attacks that impact system stability or availability

---

## 🙌 Acknowledgment

Responsible disclosures that help improve the security of the app are appreciated.
