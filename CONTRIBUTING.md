# Contributing to Stand/Sit Reminder

Thanks for your interest in improving **Stand/Sit Reminder**.
This is a **closed-source project**, so contributions are limited to feedback, ideas, and reports.

---

## 📌 Contribution Model

The source code is **not publicly available**, and external code contributions are not accepted.

However, you are welcome to:

* Report bugs
* Suggest improvements
* Propose new features
* Share usability feedback

All development and implementation is handled by the project author.

---

## 🐛 Reporting Issues

If you encounter a bug:

1. Go to the **Issues** section on GitHub
2. Create a new issue
3. Include:

* A clear description of the problem
* Steps to reproduce
* Expected vs actual behavior
* Your Windows version
* Screenshots (if applicable)

---

## 💡 Feature Requests

Feature suggestions are welcome, especially if they keep the app:

* Lightweight
* Simple
* Non-intrusive

When suggesting a feature:

* Explain the problem it solves
* Describe the expected behavior
* Keep the idea focused and minimal

---

## 🚫 What Is Not Supported

To avoid confusion:

* ❌ No pull requests
* ❌ No forks for contribution
* ❌ No external code submissions
